# Invyo Technical Test

## Setup

```bash
# Clone this repository
$ git clone https://github.com/felipeLucaDeTena/invyo-technical.git

# Go into the api repository
$ cd ../api

# Install dependencies
$ npm install

# Run the local server
$ npm start


# Go into the api repository
$ cd ../app

# Install dependencies
$ npm install

# Run the local server
$ npm start

```
